      Keyboard Shortcuts

      • ← ↑ → ↓ or WASD: Pan the canvas
      • Ctrl + ← ↑ → ↓: Navigate the hierarchy (parent / child / siblings)
      • H / J / K / L: Navigate the hierarchy (left / next / previous / right)
      • + / -: Zoom in/out
      • Enter: Expand or collapse the focused node
      • F: Refocus the current selection
      • Backspace: Return to the previous view
      • ?: Show this help dialog

      Welcome to the Cybersecurity Atlas

      Use drag or the arrow keys to pan, scroll or the +/- keys to zoom, and the control + arrow keys or H/J/K/L to move between related topics. Press ? any time to open the full shortcut sheet.
